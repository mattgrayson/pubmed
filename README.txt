pubmed.py, v. 1.0

Simple utility wrapper around Bio.Entrez for searching and retrieving 
articles from PubMed via the Entrez Programming Utilities 
<http://eutils.ncbi.nlm.nih.gov/>.

Copyright (c) 2009, Matt Grayson

-----

Required: Python 2.5 or later
Required: biopython <http://biopython.org/>

-----

To install:
$ python setup.py install
